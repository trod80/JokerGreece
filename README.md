# JokerGreece

1,6,4,12,22		<--Στηλη 1 αριθμοι
14			<--Στηλη 1 Joker
2,6,9,12,16		<--Στηλη 2 αριθμοι
2			<--Στηλη 2 Joker
5,6,7,8,12,14,17	<--Στηλη 3 αριθμοι
3,4,5			<--Στηλη 3 Joker


Δεν πρεπει να υπαρχει κανενα κενο μεταξυ των στηλων.
